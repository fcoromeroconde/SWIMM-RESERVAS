📋 Saldos Iniciales — Clientes y Sesiones
ℹ️
Importa el estado actual de tus clientes con sus saldos reales de sesiones.
Si el cliente ya existe (por documento o nombre), las sesiones se suman al saldo actual.
Columnas: nombre, documento, tipo_documento, tipo, paquete, sesiones_disponibles, vigencia_hasta, sesiones_induccion, induccion_realizada, telefono, email, notas
📊 Histórico de Ocupación — Para Estadísticas
ℹ️
Importa registros históricos de ocupación para poblar las estadísticas.
El cliente puede identificarse por nombre o documento. Si no se encuentra, se crea automáticamente como cliente histórico.
Si la celda cliente va vacía, la reserva se registra sin titular (anónima).
Columnas: fecha (YYYY-MM-DD), hora (HH:MM), carril (nombre o número), estado (asistio/no_show/cancelado/reservado), cliente, tipo_sesion
